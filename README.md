# gunman
Text-based terminal shooter built with Python.

# what you can do in the game
- you can choose a player(left or right)
- choose the amount of demage(1-10)
- then your stickman will shoot the npc
- if npc survives computer chooses the damage(1-10) and attacks you
- after one of the stickmans dies the game is over
- you can play again or stop the game

# Modules Used
- random
- os
- time

## How to Run
```
git clone https://github.com/Nikakurashvili/gunman.git
cd gunman
python gunman.py
```
